== Description ==
This is the Gimi's Wordpress theme. Gimi is a group of artists whom decided to
create their first WP theme, for educational purposes. It's a simple theme, with
html5 support, and a responsive behavior. Fast to install, it supports
multilanguages strings. It is easily editable in order to users needs.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in gimi in the search form and press the 'Enter' key in your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Feel free to extend and customize, following the license rules and additional
         rules imposed by your country, culture, environment.
